Team EDUnited

Posted on August 4, 2020 at 12:00 PM

Mission

The mission of EDUnited's project is to streamline the school registration of students. This will be accomplished by using an online, secure system in order to register students without the need to pick-up, then fill out, then return hard copies of registration forms.

Project Description

The SRS is a web-based school registration system giving parent(s) and guardian(s) the ability to register their prospective students online. The SRS produces an encrypted .zip file attached to an email to a designated school official, e.g. registrar. This official then decrypts the .zip file, producing a CSV file that can then be integrated with the school’s SIS. The decrypted .zip file may also contain any supporting documentation uploaded by the parent or guardian.

Project Documentation

EDUnited will provide a project plan, Software Requirements Specification, Technical Design Document, Programmer's Guide, Runbook, User Guide and Test Report.
